4. 正则表达式

表达式 说明
. 任意一个字符（除换行）
\\d 一个数字，等价于 [0-9]
\\w 单词字符，等价于 [a-zA-Z0-9_]
\\s 空白字符（空格、Tab、换行）
* 前一个字符重复 0 次或多次
+ 前一个字符重复 1 次或多次
? 前一个字符重复 0 或 1 次
{n} 恰好 n 次
{a, b} 匹配前一个字符a~b次，b不写的话表示任意次
[abc] 匹配 a 或 b 或 c
[^abc] 除了 a b c 以外的字符
() 分组，提取数据用
  • 常见用法有String类的matches()方法，regex库的Pattern类和Matcher类
